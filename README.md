# CLIC Rotary Bénin - Plateforme Institutionnelle 🇧🇯

Bienvenue sur le dépôt officiel du **Comité de Liaison Inter Clubs (CLIC) du Rotary au Bénin**. 

Cette plateforme web moderne et responsive permet d'unifier la communication des Rotary Clubs, Rotaract Clubs et Interact Clubs du Bénin, tout en mettant en avant leurs actions à fort impact social.

## 🚀 Fonctionnalités principales

- **Annuaire des Clubs** : Liste dynamique et détaillée de tous les clubs du pays.
- **Gestion des Actions** : Consultation des projets terminés, en cours et à venir.
- **Soumission d'Actions** : Formulaire dédié permettant aux clubs de proposer leurs projets au CLIC.
- **Interface Administrateur** : Un tableau de bord privé pour valider les actions et gérer les actualités.
- **Responsivité Totale** : Design optimisé pour ordinateurs, tablettes et smartphones.

## 🛠️ Stack Technique

- **Frontend** : HTML5, CSS3 Moderne (Vanilla), JavaScript ES6+.
- **Backend/Database** : [Supabase](https://supabase.com/) (PostgreSQL & Auth).
- **Icônes** : FontAwesome 6.
- **Typographie** : Inter (Google Fonts).

## 📂 Structure du Projet

- `/index.html` : Page d'accueil.
- `/actions.html` : Galerie des actions.
- `/clubs.html` : Annuaire des clubs.
- `/proposer-action.html` : Formulaire de soumission.
- `/admin/` : Interface de gestion (protégée par Supabase Auth).
- `/assets/css/` : Feuilles de style (design premium & african pattern).
- `/assets/js/` : Logique applicative et intégration Supabase.

## ⚙️ Installation Locale

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/ITAINNOVATE/CLICROTARY.git
   ```
2. Ouvrez `index.html` dans votre navigateur ou via un serveur local (Live Server).
